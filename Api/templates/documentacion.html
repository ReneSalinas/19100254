Documentacion de API Gashapon.

La funcion de esta API es de simular el funcionamiento de una maquina de gashapon, que es una maquina que insertas dinero y asi obtienes una oportunidad mas para sacar un premio por lo cual hay de muchas tematicas de gashapones.

Por lo cual al darle click al boton de 5$ MAS es por una jugada mas y sacar un premio mas.


FUNCIONA CON:

Lo unico necesario para funcionar que necesitas de algo exterior sera la url de la imagen del premio a desear.


CRUD

*CREATE o INSERTAR

La formula para realizar la peticion de INSERTAR sera de esta forma, estas insertando un nuevo premio a la API solo escribiendo su nombre, descripcion y la url de la imagen:

POST:

https://OrnateBitesizedCore.revis8466.repl.co/agregaPremio

BODY:

Json Content:

{

"nombre":"zoro",
"descripcion":"zoro todo chikito todo ciego",
"urlpremio":"https://m.media-amazon.com/images/I/41TLriknCBL.jpg"

}

200:

Publicado exitoso


*READ o LEER

La formula para realizar la peticion de LEER sera de esta forma, estas realizando esto para poder ver o consultar una lista completa actualizada de la base de datos de premios:

GET:

https://OrnateBitesizedCore.revis8466.repl.co/listadepremios

200:

Regresa lista de premios o productos.


*UPDATE(PATCH) o ACTUALIZAR

La formula para realizar la peticion de PATCH sera de esta forma,estas haciendo esta peticion para editar algun dato del premio:

PATCH:

https://OrnateBitesizedCore.revis8466.repl.co/editar

BODY:

Json Content

{
"idproducto":9,
"nombre":"luffy cabezon mini",
"descripcion":"luffy pero cabezon bonito",
"urlpremio":"https://cf.shopee.com.mx/file/f5d79fc18bd306f3b7d415c44972b6e8"
}

200:

Producto Actualizado.


*DELETE o ELIMINAR

La formula para realizar la peticion de ELIMINAR sera de esta forma, realizas esta peticion para eliminar de la base de datos el registro que quieres:

DEL:

https://OrnateBitesizedCore.revis8466.repl.co/eliminar

BODY:

Json Content

{
"idproducto":14
}

200:

PRODUCTO eliminado.
